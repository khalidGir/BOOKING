Widget Settings

CORS / CSP Whitelist

To test the widget on this page, make sure your business's corsWhitelist includes your test domain (e.g., localhost or 127.0.0.1).

Set it via: PATCH /api/v1/admin/business with { "corsWhitelist": ["localhost", "127.0.0.1"] }

Embed Script

Copy this snippet and paste it into your client's website <body> tag.

<script src="http://localhost:3000/widget.js" data-business="demo" data-theme="light"></script>

Inline Container

For inline mode, add a container element where the widget should render:

<div id="booking-widget"></div>
